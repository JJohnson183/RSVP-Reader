# RSVP Fast Reader

A speed reading web app using the Rapid Serial Visual Presentation (RSVP) technique to help read text content faster. Used as a first NextJs project.

## Features

- **RSVP Display**: Words presented one at a time with highlighted center character for optimal eye positioning
- **Playback Controls**: Play, pause, and restart reading sessions
- **Navigation**: Skip forward/backward one word at a time
- **Speed Control**: Adjustable reading speed (50-1000 WPM)
- **Word Counter**: Track progress and jump to any word in the text
- **Dark Reading Mode**: Optimized contrast for comfortable and focused reading
- **Responsive Design**: Works on desktop and mobile devices

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/JJohnson183/RSVP-Reader.git
   cd rsvp-reader
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
